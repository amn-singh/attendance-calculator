# 📊 Attendance Calculator

A simple and responsive Attendance Percentage Calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Calculate attendance percentage instantly
- Shows status (✅ Safe / ⚠️ Warning / ❌ Low)
- Predicts how many classes you can miss
- Clean dark UI design
- Easy to use

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
